## Install

```sh
$ yarn --frozen-lockfile
```
or
```sh
$ npm install
```

## Run

```sh
$ yarn dev
```

## Commands

### Run linter
```sh
$ yarn test:lint
```

### Run tests
```sh
$ yarn test:mocha
```

### Run in debug mode
```sh
$ yarn dev:inspect
```

### Build for production
```sh
$ yarn build
```

### Run in production mode
```sh
$ yarn start
```

### Run dev console locally
```sh
$ yarn dev:console
```
